　　企业文化是一个企业的灵魂，它折射一个企业的口味和境界，蕴含在每位企业员工的心头，蕴含在企业成长每一个脚印，蕴含在时代跳动的脉搏。我们设计企业体系的目标，是打造一个文化性企业、一个生命型企业，因此遵循以下原则：

1、全体员工共同参与

2、贴近时代、贴近企业现状及发展

3、彰显品位与品牌


（一） 《企业宗旨》--致力于民族科技之发达

　　致达者，唯民族科技发达之致力焉。直达集团自创业之初，即以发展民族科技为己任，自主创新，业达天下，企业之力深深熔铸在民族生命力、创造力、和凝聚力之中。虽然历经艰难险阻，但十年不辍，矢志不渝，是有今日直达高科技企业致欣欣气象。


（二）《企业精神》——— 致高 致强 达天下

　　所谓“站得高，看得远”，“致高”者，目光远大，高瞻远瞩。登高而俯仰自如之意也；“致强”取自强不惜。气势非凡、强大而挥斥方之意也；而《孟子 尽心上》有云：“亲亲，仁也。敬长，义也。无他，达之天下也”最后一句“无他，达之天下也”，是说为政的关键处就是将亲亲之仁与敬长之义作为创制立事的基本原则，致达集团取其意，作为创业守业之本，固有“达天下”之企业精神。
